# E-Commerce Purchase Prediction

This project builds a machine learning pipeline to predict customer purchasing behavior using an e-commerce dataset.

The project demonstrates a complete data science workflow including preprocessing, feature engineering, and model training.

## Workflow

1 Data cleaning  
2 Missing value imputation  
3 Feature engineering  
4 Feature preprocessing  
5 Machine learning model training  
6 Model evaluation

## Machine Learning Pipeline

The project uses scikit-learn pipelines to build a robust preprocessing and modeling workflow.

Components include:

• SimpleImputer  
• OneHotEncoder  
• StandardScaler  
• ColumnTransformer  
• Pipeline

## Dataset Files

E-Commerce-Sales-Data.enhanced_clean.csv  
Cleaned dataset used as input for modeling.

E-Commerce-Sales-Data.imputed_working.csv  
Dataset after missing value imputation.

impute_report_stepwise.csv  
Report documenting the imputation process.
