# E-Commerce Customer Behavior Analysis

This project performs exploratory data analysis and statistical investigation of an e-commerce customer dataset.

The objective is to understand purchasing behavior and identify relationships between customer attributes and purchasing patterns.

## Workflow

1. Data loading
2. Data cleaning
3. Feature engineering
4. Exploratory data analysis
5. Statistical testing

## Statistical Tests Used

• Independent t-test  
• Chi-square test

## Libraries

Pandas  
NumPy  
Matplotlib  
SciPy  

## Dataset Files

E-Commerce-Sales-Data.csv  
Raw dataset.

E-Commerce-Sales-Data.enhanced_clean.csv  
Processed dataset generated after data cleaning and feature engineering.
